# 🏺 فروشگاه بهشت فیروزه‌ای - پروژه کامل فروشگاهی

## 📌 معرفی پروژه
یک فروشگاه آنلاین کامل برای فروش فیروزه و زیورآلات با قابلیت‌های:
- نمایش محصولات با فیلتر و جستجو
- سبد خرید پیشرفته با محاسبه خودکار قیمت
- سیستم ورود و ثبت‌نام کاربران با اعتبارسنجی
- پنل مدیریت کامل (مدیریت محصولات، سفارشات، کاربران، تخفیف‌ها، خبرنامه)
- سیستم نظرات و امتیازدهی به محصولات
- سیستم تخفیف‌ها و کد تخفیف
- تاریخچه سفارشات کاربران
- خبرنامه (نیوزلتر)
- درگاه پرداخت سمبل
- طراحی واکنش‌گرا (مناسب موبایل و دسکتاپ)
- انیمیشن‌های حرفه‌ای
- دارک/روشن مود

## 🛠️ تکنولوژی‌های استفاده شده
### فرانت‌اند
- HTML5
- CSS3 (با متغیرها، انیمیشن‌ها، گرید و فلکس)
- JavaScript (خالص - بدون کتابخانه)
- Font Awesome (آیکون‌ها)
- Google Fonts (Vazirmatn)

### بک‌اند (سمبل)
- JavaScript
- localStorage (ذخیره‌سازی اطلاعات در مرورگر)
- سیستم احراز هویت با JWT (سمبل)

## 📋 قابلیت‌های کلیدی

### فروشگاه (index.html)
- ✅ اسلایدر بنر با حرکت خودکار
- ✅ نمایش محصولات با قیمت و تصویر
- ✅ فیلتر بر اساس دسته‌بندی
- ✅ جستجوی پیشرفته
- ✅ سبد خرید با قابلیت افزودن/حذف
- ✅ محاسبه خودکار قیمت کل
- ✅ اعمال کد تخفیف
- ✅ سیستم نظرات و امتیازدهی
- ✅ پروفایل کاربری با ویرایش اطلاعات
- ✅ تاریخچه سفارشات
- ✅ خبرنامه
- ✅ درگاه پرداخت سمبل
- ✅ دارک/روشن مود

### پنل مدیریت (admin.html)
- ✅ داشبورد با آمار فروش
- ✅ مدیریت محصولات (افزودن، ویرایش، حذف)
- ✅ مدیریت سفارشات (تغییر وضعیت، حذف)
- ✅ مدیریت کاربران (مشاهده، حذف)
- ✅ مدیریت تخفیف‌ها (افزودن، حذف)
- ✅ مدیریت خبرنامه (مشاهده، حذف)

### سیستم کاربری
- ✅ ثبت‌نام با اعتبارسنجی
- ✅ ورود به حساب کاربری
- ✅ خروج از حساب
- ✅ ویرایش پروفایل
- ✅ تغییر رمز عبور
- ✅ ذخیره اطلاعات کاربران در localStorage

## 🚀 نحوه اجرا
1. فایل index.html رو در مرورگر باز کن
2. یا از Live Server در VS Code استفاده کن
3. برای دسترسی به پنل مدیریت، فایل admin.html رو باز کن
